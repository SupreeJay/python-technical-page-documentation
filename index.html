Python as a Language

Python is a high-level programming language that has been in existence since the late 1980s. Created by Guido van Rossum, Python was first released in 1991 as a successor to the ABC programming language. Van Rossum's goal was to develop a scripting language that was easy to learn and could be used for a wide range of applications.

Python's simplicity,flexibility and extensibility have made it a popular choice for various applications. Is is widely used in web development, data analysis, machine learning, automation and education. Python's versatility has also made it a favorite among developers, scientists and researchers.

Today, python is used in many industries including web development, scientific computing and data analysis. Its application include YouTube,Instagram and Pinterest, which use Python in their backend infrastructure.Overall,Python's history and usage demonstrate its flexibility, simplicity and versatility, making it a popular programming language.

Getting Started

Here are steps to get started with programming in Python

Step 1: Install Python

  1. Go to python official website and download the latest version of Python.

  2. Follow the installation instructions for your operating system(Windows, macOS or Linux)

Step 2: Choose a Text Editor or IDE

  1. A text editor(e.g Notepad++, Sublime Text) or Integrated Development Environment(IDE e.g.,Pycharm, Visual Studio Code) is needed to write Python code

  2. Choose one that suits your needs and skill level

Step 3: Set Up Your Environment

  1. Set Up your Python environment by adding the Python executable to your system's path.

  2. Verify that python is installed correctly by opening a terminal or command prompt and typing python --version

Step 4: Learn Basic Syntax, Libraries and Framework

  1. Start learning basic Python syntax, such as variables, data types, loops,conditional statements, functions and modules

  2. Familiarize yourself with popular Python libraries and frameworks

  3. Online resources like freeCodeCamp,Codecademy,W3Schools and official Python documentation can help

Step 5: Practice writing codes and building projects.

  1. Practice writing codes with platforms like Repl.it, Ideone or Google Colab

  2. Apply your python skills to real-world project from building simple Calculator to building complex projects.

Step 6: Join online communities and Stay-Up-To-Date

  1. Join online communities, such as Reddit's r/learnpython, r/Python and Stack Overflow, to connect with other Python programmers
  2. Participate in discussions, ask questions and learn from others
  3. stay current with the latest Python developments,updates and best practices

An example of a global variable is given as:

# To define a global variable, we do:
# Define a global variable

def myfunc():
# Access the global variable

Data Types

In Python, data types are used to classify and categorize data items. They define the kind of value a variable can hold and the operations that can be performed on it. Python has several built-in data types, each serving a specific purpose.
There are several data types which includes:

  • Numeric Data Types: these are data types involving numbers.
    1. int- represent integers(i.e x = 4).
    2. float- represent decimal numbers(i.e x = 32.5).
    3. complex- represent complex numbers(i.e x = 3 + 2j).

    • Example:
    a = 5
    b = 5.0
    c = 1 + 2j
    print(type(a)) #<class 'int'>
    print(type(b)) #<class 'float'>
    print(type(c)) #<class 'complex'>

  • Sequence Data Types: this data types represent ordered collections of items and include:
    1. str: represents strings, e.g., x = "Hello World".
    2. list: represents lists, e.g., x = ["apple", "banana", "cherry"].
    3. tuple:represents tuples, e.g., x = ("apple", "banana", "cherry").
    4. range: represents ranges, e.g., x = range(6).

    • Example:
    languages = ["Swift", "Java", "Python"]
    print(languages[0]) # Swift
    print(languages[2]) # Python

  • Mapping Data Types: the mapping data type represents data in key-value pairs and includes:
    1. dict:represents dictionaries, e.g., x = {"name": "John", "age": 36}

    • Example:
    capital_city = {'Nepal': 'Kathmandu', 'Italy': 'Rome', 'England': 'London'}
    print(capital_city['Nepal']) # Kathmandu

  • Set Data Types:Set data types represent unordered collections of unique items and include:
    1. set:represents sets, e.g., x = {"apple", "banana", "cherry"}.
    2. frozenset:represents immutable sets, e.g., x = frozenset({"apple", "banana", "cherry"}).

    • Example:
    set1 = {"Geeks", "For", "Geeks"}
    print(set1) # {'Geeks', 'For'}

  • Boolean Data Types:The boolean data type represents truth values and includes:
    1. bool:represents boolean values, e.g., x = True.

    • Example:
    print(type(True)) # <class 'bool'>
    print(type(False)) #<class 'bool'>

  • Binary Data Types:Binary data types represent binary data and include:
    1. bytes:represents immutable sequences of bytes, e.g., x = b"Hello".
    2. bytearray: represents mutable sequences of bytes, e.g., x = bytearray(5).
    3. memoryview: represents memory views, e.g., x = memoryview(bytes(5)).

    • Example:
    x = b"Hello"
    print(type(x)) # <class 'bytes'>
    print(type(False)) #class 'bool'

  • None Type:The None type represents the absence of a value and includes::
    1. Nonetype:Represents the None value, e.g., x = None.

    • Example:
    x = None
    print(type(x)) # <class 'NoneType'>

Control Flow

Flow control in Python refers to the order in which individual statements, instructions, or function calls are executed or evaluated. Python provides several control flow statements to manage the execution of code based on conditions, loops, and other criteria.

This consist of different types of statement used in making decision, which includes:

Conditional Statements

Conditional statements allows you to execute different blocks of code based on certain conditions. The primary conditional statements in Python are:

  • if statement: Executes a block of code if a specified condition is true.
    For example
    x = 10
    if x > 5:
       print("x is greater than 5")

  • if-else statement:Checks multiple conditions in sequence and executes the corresponding block of code for the first true condition.
    For Example
    x = 10
    if x > 10:
       print("x is greater than 10")
    elif x == 10:
       print("x is equal to 10")
    else:
        print("x is less than 10")

  • if-elif-else statement: checks multiple conditions in sequence and executes the corresponding block of code for the first true condition.
    For example
    x = 10
    if x > 10:
         print("x is greater than 10")
    elif x == 10:
        print("x is equal to 10")
    else:
        print("x is less than 10")

Transfer Statements

Transfer statements are used to alter the normal flow of execution in a program. Python provides three main transfer statements:

  • break statement:exits the nearest enclosing loop immediately.
    For example
    for i in range(10):
       if i == 5:
          break
        print(i)
  • continue statement:skips the rest of the code inside the current loop iteration and moves to the next iteration.
    For example
    for i in range(10):
        if i == 5:
          continue
          print(i)
  • pass statement:does nothing and is used as a placeholder for future code.
    For example    if i == 5:
         pass
         print(i)

Iterative Statements

Iterative statements, also known as loops, allow you to execute a block of code repeatedly as long as a specified condition is true. Python provides two main types of loops:

  • for loop: Iterates over a sequence (such as a list, tuple, or string) and executes a block of code for each item in the sequence.
    For example
    for i in range(5):
        print(i)

  • while loop:repeatedly executes a block of code as long as a specified condition is true.
    For example:
    i = 0
    while i < 5:
        print(i)
         i += 1

Functions

A function in Python is a block of reusable code that performs a specific task. Functions help in organizing code, making it more readable and reusable. They can accept inputs (arguments) and return outputs.

Creating and Calling Functions

To define a function in Python, use the def keyword followed by the function name and parentheses. Inside the parentheses, you can specify parameters. Here's a simple example:
#creating a function in python, we use the def keyword
def greet():
     print('Hello World!')
# Calling the function
greet()
In this example, the function greet() prints "Hello World!" when called.
